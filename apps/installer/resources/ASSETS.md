# Assets Guide

## App Icon (icon.png)

You need to add a PostHog logo as `icon.png` in this directory.

**Where to get the PostHog logo:**
- Official PostHog branding: https://posthog.com/brand
- GitHub: https://github.com/PostHog/posthog/tree/master/frontend/public/icons

**Requirements:**
- Format: PNG
- Size: 512x512 pixels or larger
- Transparent background recommended

For now, you can use any placeholder icon or download the PostHog hedgehog logo from their website.

## Additional Assets (Optional)

You can also add these optional assets to `src/renderer/assets/`:

### sounds/
- `start.wav` - Windows 95/98 startup sound
- `complete.wav` - Windows "tada" sound
- `click.wav` - Click sound effect

### Images
- `posthog-logo.png` - PostHog logo for sidebar
- `sidebar.png` - Custom sidebar background image

These are all optional and the installer will work without them!
